# 荒木研究室 Webサイト デザインアイデア

## プロジェクト概要
大阪大学大学院工学研究科 荒木研究室の公式Webサイト。陽電子消滅法を用いた材料解析と次世代材料の機能設計に関する研究を紹介。

## Idea 1: Scientific Precision（科学的精密主義）
<probability>0.08</probability>

### Design Movement
**Swiss International Style × Scientific Visualization** - スイス・インターナショナルスタイルの厳格なグリッドシステムと、科学的データビジュアライゼーションの美学を融合。

### Core Principles
1. **数学的グリッド**: 黄金比と8pxベースのモジュラーグリッドによる厳格なレイアウト
2. **データの可視化**: 抽象的な装飾ではなく、実際の科学データに基づくビジュアル要素
3. **階層的明瞭性**: 情報の重要度に応じた明確な視覚的階層
4. **機能的ミニマリズム**: すべての要素に意味と目的を持たせる

### Color Philosophy
- **Primary**: Deep Navy (#0A1628) - 知性、信頼、深い探求を象徴
- **Secondary**: Silver (#C0C7D1) - 金属材料、精密機器を連想
- **Accent**: Electric Blue (#3B82F6) - 科学的発見、エネルギーを表現
- **Background**: Off-White (#F8FAFC) - クリーンルームの清潔感

### Layout Paradigm
左右非対称の2カラムレイアウト。コンテンツは左寄せで、右側に余白を大きく取り、科学論文のような読みやすさを実現。

### Signature Elements
1. **原子格子パターン**: 背景に薄く配置される結晶構造のSVGパターン
2. **データライン**: セクション間を繋ぐ細い線（回路基板を想起）
3. **ガンマ線モチーフ**: 陽電子消滅を象徴する放射状の線

### Interaction Philosophy
精密機器の操作感を再現。ホバー時は微細な変化、クリック時は確実なフィードバック。

### Animation
- ページ遷移: フェードイン（300ms ease-out）
- スクロール: パララックス効果は控えめに
- ホバー: 0.2秒の滑らかなトランジション

### Typography System
- **Display**: Noto Sans JP Bold (見出し)
- **Body**: Noto Sans JP Regular (本文)
- **Accent**: IBM Plex Mono (数値、専門用語)
</response>

## Idea 2: Material Essence（素材の本質）
<probability>0.06</probability>

### Design Movement
**Brutalist Minimalism × Material Design** - ブルータリズムの生々しさと、マテリアルデザインの物理的リアリティを組み合わせた、素材研究室にふさわしいアプローチ。

### Core Principles
1. **素材感の表現**: 金属、結晶、合金の質感をデジタルで再現
2. **構造的誠実さ**: 装飾を排し、コンテンツの構造そのものを見せる
3. **重量感と存在感**: 軽薄なUIではなく、研究の重みを感じさせる
4. **コントラストの活用**: 明暗、大小、動静の対比で視覚的インパクト

### Color Philosophy
- **Primary**: Graphite (#1C1C1E) - 炭素、鉄鋼の深い色
- **Secondary**: Titanium (#8E8E93) - チタン合金の光沢
- **Accent**: Copper (#B87333) - 銅の温かみ、伝導性を象徴
- **Highlight**: Plasma Blue (#00D4FF) - プラズマ、高エネルギー状態

### Layout Paradigm
大きなブロック要素を積み重ねる垂直スタッキング。各セクションは独立した「材料サンプル」のように扱う。

### Signature Elements
1. **金属テクスチャ**: ヘッダーやカードに微細なメタリックグレイン
2. **切断面エフェクト**: セクション境界に斜めのカット
3. **結晶構造アイコン**: カスタムアイコンセット

### Interaction Philosophy
物理的な重みを感じさせる。ボタンは押し込む感覚、カードは持ち上がる感覚。

### Animation
- ホバー: 微細な浮き上がり（translateY -2px, shadow増加）
- スクロール: 要素が下から滑り込む（staggered animation）
- ローディング: 結晶が成長するようなアニメーション

### Typography System
- **Display**: Zen Kaku Gothic New Black (力強い見出し)
- **Body**: Zen Kaku Gothic New Regular (読みやすい本文)
- **Technical**: JetBrains Mono (コード、数式)
</response>

## Idea 3: Quantum Elegance（量子的優雅さ）
<probability>0.07</probability>

### Design Movement
**Neo-Minimalism × Quantum Aesthetics** - 量子力学の不確定性と確率的美学を、洗練されたミニマリズムで表現。

### Core Principles
1. **空間の活用**: 余白を「可能性の空間」として積極的に使用
2. **確率的要素**: 微細なランダム性を取り入れた有機的なデザイン
3. **波動と粒子の二重性**: 流れるような曲線と精密な点の共存
4. **観測者効果**: ユーザーの操作で変化するインタラクティブ要素

### Color Philosophy
- **Primary**: Midnight Blue (#0F172A) - 宇宙、無限の可能性
- **Secondary**: Pearl White (#F1F5F9) - 純粋、清潔
- **Accent**: Quantum Violet (#8B5CF6) - 高エネルギー状態、神秘性
- **Glow**: Positron Cyan (#22D3EE) - 陽電子の軌跡

### Layout Paradigm
フルスクリーンセクションを基本とし、各セクションが独立した「観測窓」として機能。水平スクロールや非線形ナビゲーションも検討。

### Signature Elements
1. **波動関数グラフィック**: 背景に流れる確率波のアニメーション
2. **粒子トレイル**: マウスに追従する微細な光の粒子
3. **グロー効果**: 重要な要素に柔らかな発光

### Interaction Philosophy
量子的な「観測」を模倣。ホバーで要素が「確定」し、離れると「揺らぐ」。

### Animation
- 背景: 常に微細に動く波動パターン
- 遷移: モーフィングとフェードの組み合わせ
- スクロール: スムーズスクロールとスナップの併用

### Typography System
- **Display**: Shippori Mincho (伝統と革新の融合)
- **Body**: Noto Sans JP (可読性重視)
- **Accent**: Space Grotesk (未来的な数字)

## 選択: Idea 1 - Scientific Precision（科学的精密主義）

### 選択理由
荒木研究室の「硬派なサイエンス」という特徴と、構成案で指定された「ミニマリズム」「ディープブルー基調」「物理的事実の美しさ」という要件に最も合致するため。

### 実装方針
1. Deep Navy (#0A1628) を基調色として使用
2. Noto Sans JP をメインフォントとして採用
3. 厳格なグリッドシステムで情報を整理
4. 科学的ビジュアル（結晶構造、陽電子消滅図解）を効果的に配置
5. 控えめだが洗練されたアニメーションで専門性を演出
